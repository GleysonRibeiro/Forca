# 🎮 Jogo da Forca - Laboratório de OO

Projeto desenvolvido para fins acadêmicos como parte da disciplina de **Laboratório de Orientação a Objetos**. O objetivo principal foi aplicar conceitos avançados de POO e organização de software em uma aplicação funcional em Java.

## 🛠️ Especificações Técnicas

* **Linguagem:** Java
* **Arquitetura:** MVC (Model-View-Controller) - Garantindo a separação entre a lógica do jogo, o controle de estados e a interface com o usuário.
* **Padrões de Projeto:** Implementação de *Design Patterns* para solução de problemas de design de objetos e fluxo de dados.
* **Paradigma:** Orientação a Objetos com foco em encapsulamento e polimorfismo.

## 📝 Funcionalidades

- Gerenciamento de palavras e categorias.
- Controle de tentativas e estados do jogo (Vitória/Derrota).
- Interface interativa via console.
- Lógica de pontuação e validação de caracteres.

## 🚀 Diferenciais do Projeto

Este repositório vai além da lógica simples de um jogo, focando em:
1. **Organização de Código:** Estrutura de pacotes seguindo o padrão MVC.
2. **Extensibilidade:** Código preparado para adição de novas funcionalidades ou interfaces sem impacto na lógica central.
3. **Qualidade Acadêmica:** Desenvolvimento orientado pelas melhores práticas discutidas em laboratório.
